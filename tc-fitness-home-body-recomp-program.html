TC Fitness · Progress System

The Progression Rule

When an exercise feels manageable for 2 sessions in a row — upgrade it.

Choose ONE only:

Add Reps
Add 2 reps per set
Add a Set
Go from 3 sets to 4 sets
Slow Tempo
3-second lowering phase
Heavier Load
Increase dumbbell weight
Upgrade Variation
Move to harder exercise variation
Coaching Cue — Important: Only ONE change at a time. Never stack upgrades in the same session.
TC Fitness · Session Blueprint

Workout Structure

Exercises Per Session
5–6 dumbbell movements
Sets Per Exercise
3 working sets
Rep Range
6–12 reps (main lifts)
Effort Level
Stop 1–2 reps before failure
Effort Explained: Leaving 1–2 reps in reserve means you could do more but choose not to. This builds strength sustainably without burning you out between sessions.
TC Fitness · Movement System

Dumbbell Exercise Library

Each movement is listed with its primary dumbbell version, a bodyweight regression, and optional band fallback (only if no dumbbells available). Follow the progression ladder when you earn an upgrade.

01SQUAT PATTERN — Dumbbell Goblet Squat
3 × 8–12
RegressionBodyweight squat to chair
ProgressionsPause goblet squat → Heavy goblet squat → Split squat
"Sit between your hips, chest tall."
02HINGE PATTERN — Dumbbell Romanian Deadlift
3 × 8–12
RegressionBodyweight hip hinge
ProgressionsSingle-leg DB RDL → 3-sec slow lowering → Heavier dumbbells
"Hips back, feel hamstrings stretch."
03PUSH — Dumbbell Floor Press / Push-up
3 × 6–12
RegressionIncline push-up
ProgressionsSlow tempo push-up → Heavier dumbbell press → Decline push-up
"Control the weight, don't rush."
04PULL — One-Arm Dumbbell Row
3 × 8–12
RegressionBodyweight towel row (supported)
Band FallbackBand row (only if no dumbbells available)
Progressions2-sec pause at top → Slow eccentric → Higher reps (12–15)
"Pull elbow toward hip."
05SHOULDERS — Dumbbell Overhead Press
3 × 6–10
RegressionSeated light dumbbell press
ProgressionsSingle-arm press → Half-kneeling press → Heavier load
"Ribs down, press straight overhead."
06SINGLE LEG — Dumbbell Split Squat / Reverse Lunge
3 × 8–12 / leg
RegressionBodyweight split squat hold
ProgressionsBulgarian split squat → Slower tempo → Heavier dumbbells
"Control every inch of the movement."
